RommateOrganizerApp – Roommate Organizer Android App
A native Android application for shared household organization, built with Kotlin and XML layouts.

Overview
RommateOrganizerApp helps people living together manage shared tasks and responsibilities. Developed as a personal Android project to strengthen Kotlin development and XML UI design skills.

Tech Stack
Language: Kotlin

UI: XML Layouts (Android Views)

Platform: Android (Android Studio)

Version Control: Git

Features
Task assignment and tracking for roommates

Shared responsibility management

XML-based multi-screen UI with Kotlin logic

Clean and simple app structure for household coordination

Purpose
Built as a personal project to practice real-world Android app development using Kotlin and XML, with a focus on practical use-case design and structured UI implementation.
